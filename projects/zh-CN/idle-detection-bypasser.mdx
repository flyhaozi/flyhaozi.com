## 绕过 Idle Detection
本用户脚本会给 Idle Detection API 的调用者一个虚假的 active 响应

因为可以直接拒绝权限请求，正常情况下你并不需要这个脚本

在不给权限网站就拒绝服务时，这个脚本就派上用场了

## 示例 - 一个在线学习计时器
可以用这个示例来测试脚本是否起作用，启用脚本后计时器不会暂停
